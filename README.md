# 📊 Amazon Sales Data Analysis

## 📌 Project Overview
Analyzed Amazon sales dataset to identify revenue trends, top-performing categories, and impact of discounts.

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

---

## 📈 Key Insights
- Revenue shows monthly trends and seasonality
- Discounts have weak negative correlation with revenue
- Few top products contribute majority of sales
- Certain categories dominate overall revenue
